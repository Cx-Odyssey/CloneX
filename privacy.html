Privacy Policy

Last Updated: December 2024
Effective Date: December 2024

1. Overview

CX Odyssey ("we", "our", "the Game") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and protect your information when you play our Telegram-based space mining game.

2. Information We Collect

Data Type Purpose Source
Telegram User ID User identification and progress tracking Telegram WebApp API
Username Display in leaderboards and social features Telegram Profile
First/Last Name Personalized game experience Telegram Profile
Game Progress Save game state and statistics Game Interactions
TON Wallet Address Future token features (optional) User Connection

3. How We Use Your Information

Your information is used exclusively for:

  • Game Functionality: Saving progress, tracking statistics, enabling multiplayer features
  • Leaderboards: Displaying rankings and achievements
  • Future Features: CX Token airdrops and blockchain integration (when available)
  • Technical Support: Debugging and improving game performance
We DO NOT:
  • Sell or share your data with third parties
  • Use your data for advertising
  • Access your wallet private keys
  • Track you outside the game

4. Data Storage and Security

Where Your Data is Stored:

  • Primary Storage: Supabase (secure PostgreSQL database)
  • Backup Storage: Local browser storage (fallback only)
  • Processing: Vercel serverless functions

Security Measures:

  • Encrypted data transmission (HTTPS)
  • Secure database with row-level security
  • No storage of sensitive wallet information
  • Regular security updates

5. TON Wallet Integration

Wallet Privacy: Connecting your TON wallet is entirely optional and only used for future CX Token features.

When you connect a TON wallet:

  • We only store your public wallet address
  • We never access or store private keys
  • We cannot perform transactions without your explicit consent
  • You can disconnect at any time

6. Third-Party Services

The Game integrates with these trusted services:

  • Telegram: For user authentication and game delivery
  • TON Connect: For optional wallet connectivity
  • Supabase: For secure data storage
  • Vercel: For backend API services

Each service has its own privacy policy, which we recommend reviewing.

7. Data Retention

We retain your data:

  • Game Progress: Until you request deletion or 2 years of inactivity
  • Leaderboard Data: Until the end of each competitive season
  • Wallet Addresses: Until you disconnect or request removal

8. Your Rights

You have the right to:

  • Access: View all data we have about you
  • Correction: Update incorrect information
  • Deletion: Remove your data from our systems
  • Export: Receive your data in a portable format
  • Disconnect: Remove wallet connections at any time

To exercise these rights, contact us through:

  • Telegram: @Cx_odyssey_bot
  • GitHub: Project repository issues

9. Children's Privacy

The Game is designed for users 13 years and older. We do not knowingly collect personal information from children under 13. If you believe a child has provided us with personal information, please contact us immediately.

10. International Users

The Game is accessible worldwide. By using the Game, you consent to the transfer and processing of your information in accordance with this Privacy Policy, regardless of your location.

11. Changes to Privacy Policy

We may update this Privacy Policy to reflect:

  • New features or services
  • Changes in data protection laws
  • Improvements to our security practices

We will notify users of significant changes through the game interface.

12. Contact Information

For privacy-related questions or concerns:

  • Telegram: @Cx_odyssey_support
  • GitHub: CX Odyssey repository issues section
  • Game: Use the built-in feedback features

13. Compliance

This Privacy Policy is designed to comply with:
